html의 태그는 모두 중첩 가능하다.
✔ html의 기본 구조
- <!DOCTYPE html>
- html 태그 안에 head 태그, body 태그
✔ CSS 선언 방법 : 선택자 { 속성이름: 속성값; }
✔ CSS 적용하기 - head 태그 안에 style 태그 안에 작성
✔ 선택자의 종류
- HTML 태그 (최대한 안쓰는 방향으로...)
- 클래스 (선택자는 .클래스이름)
- 아이디 (선택자는 #아이디이름)

⚠️ 주의! 클래스는 한가지 이름을 여러 요소에 사용해도 괜찮지만,
아이디는 한 가지 이름은 *왠만하면* 한 가지 요소에만 사용하는 게 좋다!

4일차💻

📗 패딩과 마진을 배우고, 다양한 단위를 학습하였습니다!

✔ 여백을 설정하는 방법 = 패딩 / 마진
- 패딩 = 테두리와 텍스트 사이 여백
- 마진 = 테두리 바깥의 여백


✔ 웹 개발에서 (주로) 사용되는 단위는 px, em, rem, %
- px: 절대 값!
- em: 상위 요소의 폰트 크기를 기준으로 하는 상대값
- rem: 최상위 요소의 폰트 크기를 기준으로 하는 상대값
- %: 상위 요소를 기준으로 하는 비율값


✔ (심화) 반응형 웹
- 화면 크기에 따라 변하는 웹사이트
- 반응형 웹사이트를 도전하기 전에 기초 html, css를 마스터하자.

padding과 margin을 나타내보자구!
